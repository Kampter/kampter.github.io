Global Game Jam 2023еӣһйЎҫ

еҝғеҫ—

жҠұжӯүиҝҷд№ҲжҷҡжүҚжқҘеҶҷиҝҷдёӘеӣһйЎҫпјҢиҝ‘жңҹеҝҷдәҺиҝҮе№ҙпјҢжғ…дәәиҠӮпјҢд»ҘеҸҠеӯҰд№ зӯүзӯүдәӢйЎ№пјҢе№ҙеҗҺеҲ°зҺ°еңЁз»ҲдәҺжңүдёҖж®өж—¶й—ҙзӣёеҜ№жҜ”иҫғе®үйқҷеҸҜд»ҘеӣһйЎҫдёҖдёӢиҝҷдёӘиөӣдәӢгҖӮ

йҰ–е…Ҳж„ҹи°ўGGJ 2023иғҪеңЁиҝҷдёӘж—¶жңҹеҸ¬ејҖпјҢжҲ‘д»ҺеүҚе…¬еҸёзҰ»иҒҢеҲ°зҺ°еңЁеӯҰд№ жёёжҲҸеҲ¶дҪңе·Із»ҸжңүеҚҠе№ҙеӨҡзҡ„ж—¶й—ҙгҖӮдёҖзӣҙеңЁеӯҰд№ дҪҶжҳҜжІЎжңүд»Җд№ҲжңәдјҡиғҪеӨҹеӣһйЎҫеӯҰдјҡд»Җд№ҲпјҢиҝҳж¬ зјәд»Җд№ҲпјҢиҝҳжғіиҰҒд»Җд№ҲгҖӮиҖҢдё”еңЁиҝҷз§ҚжҜ”иөӣзҡ„еҺӢеҠӣдёӢпјҢеҫҲеӨҡй—®йўҳжүҚдјҡжҡҙжјҸеҮәжқҘпјҢж„ҹи°ўиҝҷж¬ЎиғҪеӨҹи®©жҲ‘жҖ»з»“зҡ„жңәдјҡгҖӮ

е…¶ж¬Ўж„ҹи°ўжҲ‘зҡ„з»„е‘ҳпјҡ48е°Ҹж—¶ејҖеҸ‘иҫӣиӢҰдәҶпјҢз»„еҶ…зҡ„йҷӨдәҶжҲ‘д№ӢеӨ–йғҪжҳҜе·Із»ҸеңЁжёёжҲҸе…¬еҸёеҶ…е·ҘдҪңзҡ„еӨ§дҪ¬пјҢеҫҲй«ҳе…ҙиғҪдёҺдҪ д»¬дёҖиө·еҗҲдҪңе®ҢжҲҗиҝҷдёӘдҪңе“ҒпјҢеёҢжңӣжҲ‘жІЎжңүз»ҷдҪ д»¬жӢ–еҗҺи…ҝгҖӮ

жңҖеҗҺж„ҹи°ўжҲ‘зҡ„еҘіжңӢеҸӢпјҢж„ҹи°ўеҘ№еңЁејҖеҸ‘еүҚдёҺжҲ‘дёҖиө·еҮҶеӨҮпјӣејҖеҸ‘дёӯйҷӘдјҙжҲ‘е№¶йј“еҠұпјҢеё®жҲ‘дёҖиө·еҜ»жүҫзҒөж„ҹпјӣз»“жқҹеҗҺжҲ‘д»¬дёҖиө·еҲҶдә«е–ңжӮҰгҖӮеҘ№д№ҹе–ңж¬ўжёёжҲҸпјҢд№ҹе–ңж¬ўе®«еҙҺйӘҸзҡ„еҠЁз”»гҖӮжёёжҲҸеңәжҷҜеҶ…зҡ„йғЁеҲҶжёІжҹ“йЈҺж јд№ҹжҳҜжҲ‘зҡ„дёҖзӮ№з§ҒеҝғпјҢеёҢжңӣжңӘжқҘжңүжңәдјҡиғҪжһ„е»әдёҖдёӘзӢ¬еұһдәҺжҲ‘д»¬зҡ„дё–з•ҢгҖӮ

жҠҖжңҜе®һзҺ°

йүҙдәҺиҝҷдёӘ48е°Ҹж—¶зҡ„е°ҸйЎ№зӣ®пјҢиҷҪз„¶д№ӢеүҚдёҖзӣҙеңЁзҺ©HDRPжёІжҹ“з®ЎзәҝгҖӮиҝҷж¬Ўе°ұиҖҒиҖҒе®һе®һз”ЁURPдәҶпјҢиҝҷд№ҹжҳҜжҲ‘з¬¬дёҖж¬Ўз”ЁURPгҖӮд№ӢеүҚеңЁHDRPдёӯеӨ§йғЁеҲҶshaderйғҪз”Ёshader graphе°ұеҸҜд»Ҙе®ҢжҲҗпјҢиҝҷж¬ЎеңЁurpдёӯзәҜжүӢеҶҷиҝҳжҳҜеҫҲжңүжҢ‘жҲҳжҖ§зҡ„гҖӮ

ж•ҙдҪ“зҫҺжңҜйЈҺж јпјҢеёҢжңӣе°қиҜ•дёҖдёӢеҚЎйҖҡжёІжҹ“гҖӮеңәжҷҜжёІжҹ“еёҢжңӣеҖҹйүҙеҗүеҚңеҠӣйЈҺж јжІ»ж„ҲгҖҒе”ҜзҫҺзҡ„ж„ҹи§үпјҢеҗҢж—¶еҖҹйүҙе®«еҙҺйӘҸеҠЁз”»дёӯеҜ№иҮӘз„¶е’Ңе№ізҡ„иҝҪжұӮгҖӮиҖҢдәәзү©жҲ‘д»¬йҖүжӢ©ж јйІҒзү№иҝҷдёҖз»Ҹе…ёжј«еЁҒи§’иүІпјҢжҲ‘жғіе°қиҜ•дёҖдёӢеҺҡж¶ӮйЈҺж јеёҰжқҘзҡ„з«ӢдҪ“ж„ҹгҖӮ

еҹәзЎҖеңәжҷҜжҗӯе»ә

еҲҡеҘҪеңЁ2022е№ҙеә•иҠұдёҚе°‘зІҫеҠӣеӯҰд№ е’ҢжҺ’еқ‘зҡ„houdini PDGжөҒзЁӢиҝҷж¬ЎеҲҡеҘҪеҸҜд»Ҙеә”иҜҘз”ЁеңЁиҝҷж¬ЎйЎ№зӣ®дёӯгҖӮ

з”Ёhoudini з”ҹжҲҗHeightmapе·Із»ҸжҳҜйқһеёёиҪ»жқҫж„үеҝ«зҡ„дәӢжғ…дәҶпјҢиҝҷдёӘйЎ№зӣ®дёҚйңҖиҰҒй«ҳзІҫеәҰйӣ•зҗўпјҢз®ҖеҚ•и°ғж•ҙдёҖдәӣз»ҶиҠӮе°ұеҸҜд»Ҙжҗӯе»әPDGиҠӮзӮ№дәҶгҖӮе“ҰеҜ№пјҢеңЁиҝҷд№ӢеүҚиҰҒеңЁunityдёӯеҮҶеӨҮеҘҪterrain layerд»ҘеҸҠйҖҡиҝҮйҡҸеҚіж’’зӮ№з§ҚжӨҚзҡ„жӨҚзү©prefabпјҢеҗҢж—¶и®ҫзҪ®еҘҪLODгҖӮ

еӨ§иҮҙжөҒзЁӢе°ұжҳҜеңЁhoudiniдёӯиҺ·еҸ–unityдёӯеҮҶеӨҮеҘҪзҡ„prefab pathпјҢеҜјеҮәиҮіfoliage.csvдёӯпјҢз„¶еҗҺжҠҠunity_instanceеҶҷе…Ҙprefab pathеұһжҖ§еҶҚеҜје…ҘunityдёӯcookгҖӮ

з”ЁattributewrangleиҜ»еҸ–csvж–Үд»¶дёӯзҡ„и·Ҝеҫ„е№¶еҶҷе…Ҙж’’зӮ№зҡ„s@unity_instanceеұһжҖ§дёӯпјҢиҝҷдёҖжӯҘж“ҚдҪңеҫҲеғҸд№ӢеүҚеңЁjupyterдёӯж“ҚдҪңcsvж–Үд»¶гҖӮеҸҜд»ҘеҠ дёҖжӯҘе°Ҷposition, normal, orient, pscaleзӯүеұһжҖ§еҜјеҮәPcacheпјҢйҖҡиҝҮVFX graphеҲ¶дҪңеҸҜдәӨдә’зҡ„иҚүең°пјҲе®ҳж–№жј”зӨәDemoпјүиҝҷж¬ЎжІЎжңүдҪҝз”ЁгҖӮ

жҲҗе“Ғж•ҲжһңеӨ§иҮҙжҳҜиҝҷж ·пјҢеҝҪз•ҘиҝҷдёӘеҫҲеғҸueзҡ„еңәжҷҜпјҢжҳҜECS2 input systemиҮӘеёҰзҡ„гҖӮ

иҚүең°жёІжҹ“

зӣ®еүҚжҲ‘жҜ”иҫғе–ңж¬ўзҡ„жңүдёүз§Қж–№ејҸжқҘеҲ¶дҪңиҚүең°гҖӮ

  • з¬¬дёҖз§ҚжҳҜterrainз¬”еҲ·еҲ·дёҠеёҰLODзҡ„prefabжЁЎеһӢпјҢиҝҷжҳҜжёІжҹ“ж•ҲжһңжңҖеҘҪзҡ„ж–№жЎҲпјҢд№ҹжҳҜжҲҗжң¬жңҖй«ҳзҡ„гҖӮ
  • з¬¬дәҢз§ҚжҳҜterrainз¬”еҲ·еҲ·дёҠеёҰLODзҡ„prefabжҸ’зүҮиҚүең°пјҢж•ҲжһңеҜ№жҜ”з¬¬дёҖз§Қз•ҘжҳҫзІ—зіҷдҪҶжҳҜжҖ§иғҪжҸҗй«ҳеҫҲеӨҡгҖӮ
  • з¬¬дёүз§ҚжқҘиҮӘе®ҳж–№зҡ„GrassWind DemoпјҢеңЁVFX graphдёӯдҪҝз”ЁеҲҡеҲҡhoudiniеҜјеҮәзҡ„PcacheпјҢиҖҢжЁЎеһӢзҡ„йғЁеҲҶеҸҜд»Ҙд»»йҖүжЁЎеһӢжҲ–иҖ…жҸ’зүҮпјҢеҸӘжҳҜжҖ§иғҪдёҠзҡ„еҢәеҲ«гҖӮ

иҝҷж¬Ўж–№жЎҲжҲ‘д»¬йҖүжӢ©дәҶз¬¬дәҢз§ҚпјҢеёҰLODзҡ„жҸ’зүҮиҚүең°гҖӮжҸ’зүҮиҚүең°зҡ„еҲ¶дҪңд№ҹеҫҲз®ҖеҚ•пјҢеңЁhoudiniдёӯз®ҖеҚ•и°ғж•ҙдёҖдёӢдҪҚзҪ®пјҢеһӮзӣҙзӣёжңәжӢҚж‘„еҮәtextureеҚіеҸҜгҖӮжіЁж„Ҹдј йҖ’зҗғеҪўжі•зәҝе№¶дё”еҸӘдҝқз•ҷеҗ‘дёҠзҡ„йғЁеҲҶгҖӮ

з„¶еҗҺз”Ёtriangulate2d еӢҫеӢ’еҮәиҫ№з•Ңе№¶дё”з»„еҗҲжҲҗжҸ’зүҮиҚүжЁЎеһӢгҖӮиҚүең°жЁЎеһӢеҜје…ҘunityеҗҺе…ій—ӯжҠ•е°„йҳҙеҪұпјҢйҷҚдҪҺйҳҙеҪұеёҰжқҘзҡ„жқӮд№ұж„ҹгҖӮ

иҚүең°жёІжҹ“shader жҲ‘иҝҷж¬ЎдҪҝз”Ёshader graphжқҘеҲ¶дҪңпјҢж„ҹи°ў2021.3зүҲжң¬зҡ„urpдёӯж”ҜжҢҒshader graphгҖӮ

жҠҠиҚүд»ҺдёҠеҲ°дёӢз”Ёtop colorе’Ңbottom color lerpиө·жқҘжҺ§еҲ¶дё»дҪ“пјҢйҡҸеҗҺеҸ еҠ дёӨеұӮgradient noiseд»Һж•ҙдҪ“жҺ§еҲ¶иҚүең°зҡ„йўңиүІиҖҢдёҚд»…д»…жҳҜеҚ•дёҖжЁЎеһӢзҡ„еҸҳеҢ–еҗҢж—¶жЁЎжӢҹиҚүең°ж•ҙдҪ“йҳҙеҪұж•ҲжһңгҖӮWindзҡ„vertex movmentд№ҹжҳҜеҗҢзҗҶпјҢдҪҶжҳҜиҝҷйҮҢеҒ·жҮ’жІЎжңүеҒҡпјҢеҗҺйқўжңүжңәдјҡеҺ»е®һзҺ°дёҖдёӢгҖӮ

ж ‘жңЁжёІжҹ“

ж ‘е№Ізҡ„йғЁеҲҶз”Ёhalf lambert е’Ңblinn specularе®һзҺ°гҖӮ

ж ‘еҸ¶д№ҹдҪҝз”ЁжҸ’зүҮж ‘еҸ¶е®һзҺ°пјҢshaderйғЁеҲҶз•Ҙеҫ®еӨҚжқӮпјҢйҷӨдәҶhalf lambert е’Ңblinn specularд»ҘеӨ–гҖӮи®Ўз®—ж ‘еҸ¶иҮӘйҳҙеҪұеўһеҠ з»ҶиҠӮгҖӮ

жЁЎжӢҹжҠ•е°„ж•ҲжһңпјҢеҫҲеӨ§зЁӢеәҰдёҠжҸҗеҚҮж ‘еҸ¶йўңиүІзҡ„зәҜеәҰгҖӮ

жЁЎжӢҹеҒҮзҡ„SSSж•Ҳжһң

еұҸе№•з©әй—ҙиҫ№зјҳе…ү

жңҖеҗҺзҡ„ж•ҲжһңеӣҫеҰӮдёӢпјҢиё©еқ‘зӮ№пјҢи®°еҫ—еҶҷйҳҙеҪұpassе№¶дё”йҳҙеҪұд№ҹиҰҒalpha clipгҖӮиҝҳжҳҜжҜ”иҫғжқӮд№ұпјҢжІЎжңүжҠҠжЁЎеһӢеҜје…Ҙhoudiniдј йҖ’дёҖдёӢзҗғеҪўжі•зәҝпјҢ48е°Ҹж—¶ж—¶й—ҙжқҘдёҚеҸҠдәҶгҖӮ

дәәзү©жёІжҹ“

дәәзү©жёІжҹ“жҲ‘з”Ёзҡ„жҳҜshader graph + custom functionзҡ„ж–№ејҸе®һзҺ°гҖӮ

еӨ§иҮҙжҖқи·ҜдёҺзӣҙжҺҘжүӢеҶҷжІЎжңүеӨӘеӨ§еҢәеҲ«гҖӮйҰ–е…ҲйҒҚеҺҶеңәжҷҜеҶ…зҡ„зҒҜе…үпјҲеӨҡзҒҜе…үж”ҜжҢҒпјүиҺ·еҫ—жҜҸдёӘзҒҜе…үзҡ„йўңиүІпјҢж–№еҗ‘пјҢиЎ°еҮҸеұһжҖ§з”Ёд»ҘеҸ еҠ и®Ўз®—гҖӮе…үз…§жЁЎеһӢз”ЁsmoothstepеҮҪж•°жҺ§еҲ¶еҚҠе…°дјҜзү№жЁЎеһӢзҡ„иҫ“еҮәпјҢеҗҢж—¶и®ҫзҪ®SmoothеҸӮж•°жқҘи°ғж•ҙиҫ№з•Ңзҡ„е…үж»‘еәҰгҖӮ

жңҖе·Ұиҫ№smooth = 0, дёӯй—ҙsmooth = 0.2 еҸіиҫ№дёәејҖеҗҜиҫ№зјҳе…үзҡ„ж•ҲжһңгҖӮиҫ№зјҳе…үз”ЁйқһеёёеҒ·жҮ’зҡ„ж–№ејҸе®һзҺ°

ж°ҙйқўжёІжҹ“

з”Ёж №жҚ®ж—¶й—ҙеҸҳеҢ–зҡ„gradient noiseжҺ§еҲ¶ж°ҙйқўзҡ„vertex position з§»еҠЁпјҢдё»иҰҒз”Ёд»Ҙе®һзҺ°е°Ҹе№…еәҰзҡ„жҪ®жұҗзҡ„ж„ҹи§үгҖӮ

ж°ҙйқўйўңиүІз”ЁеұҸе№•з©әй—ҙж·ұеәҰжқҘжҺ§еҲ¶пјҢlerpж·ұж°ҙеҢәе’Ңжө…ж°ҙеҢәйўңиүІпјҢиҝҷдёӘе®һзҺ°жқҘиҮӘunityзҡ„е®ҳж–№жј”зӨәи§Ҷйў‘гҖӮ

йҖҡиҝҮи°ғж•ҙStrengthе’ҢdepthжҺ§еҲ¶жө…ж°ҙе’Ңж·ұж°ҙеҢә

еҸҚе°„е’Ңж°ҙйқўжіўзә№е®һзҺ°гҖӮйҖүеҸ–дёӨеј ж°ҙйқўжі•зәҝзә№зҗҶеӣҫпјҢи®ҫзҪ®еҘҪж №жҚ®ж—¶й—ҙз§»еҠЁзҡ„ж–№еҗ‘е’ҢйҖҹеәҰпјҢз”Ёreoriented normal blendзҡ„еҸ еҠ ж–№ејҸпјҲж•ҲжһңжӣҙеҘҪпјүпјҢз”Ёд»ҘжЁЎжӢҹж°ҙйқўиў«йЈҺеҗ№еҠЁзҡ„жіўзә№гҖӮж°ҙйқўеҸҚе°„йҖҡиҝҮжҠҠеҲҡеҲҡзҡ„з»“жһңеҸ еҠ иҮіScreen positionдёӯпјҢеҶҚжҠҠз»“жһңиҫ“еҮәиҮіScene Colorзҡ„UVдёӯ, дёҺз»ҷеҲҡеҲҡжұӮеҫ—зҡ„ж·ұеәҰеӣҫз»“жһңж №жҚ®aplhaйҖҡйҒ“иҝӣиЎҢlerpпјҢиҫ“еҮәиҮіnormalеҚіеҸҜиҺ·еҫ—иҝҳз®—дёҚй”ҷзҡ„е…·жңүеҸҚе°„ж•Ҳжһңзҡ„ж°ҙйқўжіўзә№гҖӮ

жңҖз»Ҳж•ҲжһңеҰӮеӣҫ

дҪ“з§Ҝдә‘

дә‘зӣҙжҺҘз”Ёзҡ„cubemapиҙҙеӣҫиҙҙдёҠеҺ»зҡ„skyboxпјҢ48е°Ҹж—¶е®һеңЁжІЎжңүж—¶й—ҙе®һзҺ°дәҶ
